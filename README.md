# Project
A modern, single-page portfolio website built with **HTML, CSS, and JavaScript**, designed for a **student / early-career developer**. It focuses on:

- A clean, gradient-driven UI that looks great on desktop and mobile.
- Clear sections recruiters care about: About, Projects, Skills, Timeline, Contact.
- Simple structure that looks good on GitHub and is easy to deploy via GitHub Pages.

* Tech Stack

- HTML5
- CSS (custom properties, gradients, flexbox, grid)
- JavaScript (no frameworks)
